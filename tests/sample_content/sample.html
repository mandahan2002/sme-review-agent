SAP Fiori Launchpad Navigation Guide

This guide explains how to navigate the SAP Fiori Launchpad and access business applications in SAP S/4HANA Cloud. The Fiori Launchpad is the central entry point for all SAP S/4HANA applications.

Accessing the Fiori Launchpad

Open your web browser and navigate to your company's SAP Fiori URL. Log in using your SAP user ID and password. The default Fiori Launchpad URL format is https://[server]/sap/bc/ui5_ui5/ui2/ushell/shells/abap/FioriLaunchpad.html.

Note: Internet Explorer 11 is the recommended browser for SAP Fiori applications. Other browsers may have limited functionality.

Understanding the Home Page

The Fiori Launchpad home page displays tiles organized in groups. Each tile represents a Fiori application or a classic SAP GUI transaction. Tiles can be standard (provided by SAP) or custom (configured by your administrator).

You can personalize your home page by dragging and dropping tiles to rearrange them. To add tiles, click the Edit Home Page button in the top-right corner.

Searching for Applications

Use the search field at the top of the screen to find applications. Type the application name or business function. The search returns both Fiori apps and classic transactions (SAP GUI shortcuts).

Navigation Between Applications

To navigate between open applications, use the Back button in the browser or the application's own navigation controls. The shell header shows breadcrumbs for the current navigation path.

The Me Area (user icon, top right) provides access to:

  • User profile settings
  • Notification center
  • User account and sign out

Working with Notifications

Workflow notifications appear in the notification bell icon. Click a notification to navigate directly to the relevant business object. You can configure notification preferences in the Me Area settings.

Notification delivery depends on the workflow configuration in your SAP system. Notifications are not available in all deployment scenarios.

Mobile Access

The SAP Fiori Launchpad is accessible from mobile devices using the SAP Fiori Client app. Download the app from the Apple App Store or Google Play Store. Some applications may not be fully optimized for mobile use.

Accessibility Features

SAP Fiori supports keyboard navigation. Use Tab to move between elements and Enter to activate. Screen readers are compatible with WAI-ARIA standards.

Troubleshooting

If tiles do not load, clear your browser cache and cookies, then refresh the page. If problems persist, contact your SAP system administrator.

For performance issues, ensure you are using a supported browser with JavaScript enabled. Pop-up blockers should be disabled for the SAP Fiori URL.
